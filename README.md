# SpecialPy OS Beta v3

**SpecialPy OS Beta v3** is a lightweight operating system simulation built entirely in Python, designed for experimenting, learning, and everyday productivity.

### Features

* 📝 **Text Editor** – Create and edit text files with ease.
* 💻 **Terminal** – Execute supported commands through a built-in command-line interface.
* 📁 **File Viewer** – Browse and view your files from within the OS.
* ⚡ **Fast & Lightweight** – Simple, responsive, and easy to use.
* 🛠️ **Actively Developed** – New features, improvements, and bug fixes are continuously being added.

**Version:** Beta v3

SpecialPy OS is still in beta, so you may encounter bugs or unfinished features. Feedback and suggestions are always welcome to help improve future releases.
